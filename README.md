# Food Store - TPI Programación II

Sistema de consola para gestión de pedidos de comida.

## Requisitos

- Java 21+
- NetBeans 14+ (o cualquier IDE con soporte Java)

## Ejecutar en NetBeans

1. **Abrir proyecto**: `File > Open Project` → seleccionar `TPI-FoodStore/`
2. **Compilar**: Botón derecho sobre el proyecto → `Clean and Build`
3. **Ejecutar**: `Run > Run Project` (o `F6`)

El `main()` está en `src/integrado/prog2/Main.java`.

## Ejecutar por terminal

```bash
javac -d build src/integrado/prog2/**/*.java
java -cp build integrado.prog2.Main
```
